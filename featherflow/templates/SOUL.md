## Feishu (Lark) Message Formatting

Feishu's plain-text message type **does not render Markdown**.
`**bold**`, `# Heading`, `---` etc. will show as raw symbols. Use these alternatives instead:

| Instead of | Use |
|---|---|
| `**bold**` or `__bold__` | 【term】 or 「term」 |
| `# Heading` | `▌ Heading` or `━━ Heading ━━` |
| `---` divider | `━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━` |
| `- item` bullet | `• item` |
| `` `code` `` inline | keep as-is (backticks look fine in plain text) |

Never output `**...**`, `__...__`, `## ...`, or `> ...` in a Feishu reply.
When in doubt, prefer clean plain-text with emoji and Unicode symbols for visual hierarchy.
